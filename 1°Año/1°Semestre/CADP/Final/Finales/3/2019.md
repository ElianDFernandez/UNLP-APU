    ¿Cuales son las principales diferencias entre las estructuras de control for, while() y repeat?. Enuncie un problema sencillo para cada estructura de control que justifique su uso e implemente su solucion 

    En el caso de for, tenemos una estructura de repeticion, la cual podremos usar para iterar x cantidad de veces un algoritmo sin importar ninguna condicion. Por ejemplo queremos obetner la tabla de multiplicacion de algun numero entonces haremos 
        declaraciones de variables...
        writeln('Ingrese numero');
        readln(numero);
        for i:= 1 to 10 do begin 
            writeln(numero ,' * ', i, ' = ', numero*i);
        end;
    En el caso de while, tenemos una estructura la cual evalua una condicion y siempe que esta se cumpla entrara y seguira iterando. 
    Un caso de uso de esta estructura puede ser que ingrese numero hasta que se ingrese un numero positivo
        while (numero < 0 ) do begin 
            readln(numero);
        end;
    En el case del repeat esta estructura primero ejecuta el algoritmo y luego evalua la condicion si esta es false seguira ejecutandose. 
    Podemos utilizar esta estructura para un menu donde primero debemos imprimir la opciones simepre y luego leer la opcion. por ejemplo  
        repeat 
            writln('Menu'),
            writeln('1. Avanzar');
            writeln('2. Salir');
            read(opcion);
            case opcion 
                1: 
                    avanzar();
                2: 
                    writeln('Saliendo');
            end;
        until (opcion = 2);
